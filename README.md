## TechStack

- [NextJS](https://nextjs.org/)
- [Chakra UI](https://chakra-ui.com/)
- [Formik](https://formik.org/docs/overview)
- [Yup](https://github.com/jquense/yup)
- [React Query](https://react-query.tanstack.com/)

## Getting Started

Install the deps:

```bash
yarn install
```

Then run it:

```bash
yarn dev
```
